🔬 Metodologia do Índice Compara Preço

O Índice Compara Preço é calculado mensalmente com base na variação do preço médio dos produtos monitorados em cada categoria. Comparamos o preço médio do mês atual com o mês anterior, considerando apenas produtos com histórico de pelo menos 30 dias. Os dados são coletados automaticamente via API oficial do Mercado Livre e atualizados diariamente. Uma variação negativa indica que os preços caíram (bom para o consumidor), enquanto uma variação positiva indica alta de preços.
